Information for participants of research studies

Title of the project

Function communication

Description

This study is part of a research project conducted by Eric Schulz (Harvard University) and Francisco Quiroga. We investigate how people perceive and describe functions. Please read the instructions carefully and try to describe the functions as well as you can. If you have any questions, please write an email to eric.schulz@fas.harvard.edu

Consent form

This study will ask you to perform a simple computer task in which you have to decribe functions.

This study takes around 20 minutes on average.

You will be paid $2.00 in total.
